1. install crewai using uv: `uv tool install crewai`
2. create a new project for crew: `crewai create crew <name-of-project>`
   for this you called the above like this: `F:\Agentic-AI\3_ehsan_crew> crewai create crew debate`. Doing so create the `deabte` folder under the `3_ehsan_crew` directory
3. once you have modified your agents and tasks and the crew.py and main.py programs, cd into the `<name-of-project>` and `crewai run`
